# /plan

**Deutsch:** Verwandle ein freigegebenes Design in einen testbaren Aufgabenplan,
den ein Agent ohne weiteren Kontext ausfuehren kann. Lies
`docs/process/workflow.md` (Plan) und `docs/process/journal-state-plans.md` fuer
den Plan-Ablageort. Beschreibe kleine Aufgaben mit konkreten Dateien, Code und
Testkommandos. Pflicht fuer jeden Tier-3+-Change.

**English:** Turn an approved design into a testable task breakdown that an
agent can execute without more context. Read `docs/process/workflow.md` (Plan)
and `docs/process/journal-state-plans.md` for where plans live. Write small
tasks with concrete files, code, and test commands. Required for every Tier 3+
change.
