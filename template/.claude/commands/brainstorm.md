# /brainstorm

**Deutsch:** Verwandle eine Idee oder ein Problem in ein freigegebenes Design,
bevor gebaut wird. Lies `docs/process/workflow.md` und starte bei neuen oder
unklaren Vorhaben zuerst mit `docs/process/start-here.md`. Stelle wenige
gezielte Fragen, fasse Annahmen zusammen, nenne Trade-offs und schreibe das
abgestimmte Design in eine Spec-Datei. Keine Umsetzung vor Freigabe.

**English:** Turn an idea or problem into an approved design before building.
Read `docs/process/workflow.md` and, for new or unclear projects, start with
`docs/process/start-here.md`. Ask a few targeted questions, summarize
assumptions, name trade-offs, and write the agreed design to a spec file. Do not
start implementation before approval.

Next: `/plan`.
