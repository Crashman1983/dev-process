# /quick

**Deutsch:** Nutze die Tier-1/2-Abkuerzung fuer kleine, isolierte Aenderungen
ohne separate Plan-Datei. Lies `docs/process/workflow.md` (Quick). Vor dem
Editieren: Ziel, beruehrte Dateien und Risiko nennen; fuer Tier 2 einen Test
hinzufuegen. Zu `/plan` eskalieren, sobald Persistenz, Auth, Contracts oder
externe Integrationen betroffen sind.

**English:** Use the Tier 1/2 shortcut for small, isolated changes without a
separate plan file. Read `docs/process/workflow.md` (Quick). Before editing:
state goal, touched files, and risk; add a test for Tier 2. Escalate to `/plan`
as soon as persistence, auth, contracts, or external integrations are in scope.
