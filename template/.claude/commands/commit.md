# /commit

**Deutsch:** Erstelle einen atomaren Conventional Commit. Lies
`docs/process/commits.md`. Nutze `type: imperative subject` unter 72 Zeichen,
ein logischer Change pro Commit, und dokumentiere uebersprungene Gates oder
entfallenen Scope im Body. Keine direkten Commits auf `main`; arbeite auf einem
Feature-Branch und merge erst nach Review-Gate fast-forward-only.

**English:** Record one atomic conventional commit. Read
`docs/process/commits.md`. Use `type: imperative subject` under 72 characters,
one logical change per commit, and document skipped gates or dropped scope in
the body. No direct commits to `main`; work on a feature branch and merge
fast-forward-only after the review gate.
