# Process Work / Prozessarbeit

**Deutsch:** Dieses Verzeichnis ist das repo-lokale Arbeitsgedaechtnis fuer den
Entwicklungsprozess.

- `journal/` dokumentiert, warum Entscheidungen getroffen wurden.
- `plans/` enthaelt aktive Umsetzungsplaene und archivierte Plaene.
- `state/` enthaelt branch-bezogenen Kontext und naechste Schritte.

Halte Eintraege kurz, sachlich und datiert. Git dokumentiert, was sich geaendert
hat; diese Dateien dokumentieren Begruendung und Uebergabestand.

**English:** This directory is the repository-local working memory for the
development process.

- `journal/` records why decisions were made.
- `plans/` stores active implementation plans and archived plans.
- `state/` stores branch-scoped context and next steps.

Keep entries short, factual, and dated. Git records what changed; these files
record reasoning and handoff state.
